# Havun - Bedrijfswebsite

Website voor Havun: webhosting, SEO en webapplicaties.

**Live:** https://havun.nl

## Tech Stack

- Next.js 16
- React 19
- Tailwind CSS
- TypeScript

## Lokaal draaien

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy

```bash
npm run build
# Deploy via SSH naar server
```
